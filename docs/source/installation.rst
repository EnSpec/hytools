=============
 Installation
=============


Dependencies
============

   * numpy
   * h5py
   * scipy


Installing HyTools
==================
To download to hytools simply clone the github repository

.. code-block:: bash
     
    git clone https://github.com/EnSpec/hytools.git
 

Run the following command inside the hytools folder to install

.. code-block:: bash
     
   python setup.py install
