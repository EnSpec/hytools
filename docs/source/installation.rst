=============
 Installation
=============


Dependencies
============

   * numpy
   * h5py
   * ray
     
Installing HyTools
==================
To download to hytools simply clone the github repository

.. code-block:: shell
     
   $ git clone https://github.com/EnSpec/hytools.git
 

and run the following command inside the hytools folder to install

.. code-block:: shell
     
   $ python setup.py install
